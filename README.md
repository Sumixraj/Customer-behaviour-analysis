Customer Behaviour Analysis using Microsoft Power BI

Project Overview:
This project focuses on analyzing customer transaction data to understand purchasing behavior, sales trends, and customer preferences. Using Microsoft Power BI, the raw data is transformed into interactive dashboards that help in making data-driven business decisions.

Objectives:
Analyze customer purchasing patterns
Identify top-selling product categories
Understand customer demographics
Track monthly sales trends
Provide actionable business insights

Tools & Technologies Used:
Microsoft Power BI
Power Query (Data Cleaning)
DAX (Data Analysis Expressions)
Data Visualization Techniques

Dataset Description:

The dataset contains customer transaction details such as:
Customer ID
Age
Gender
Product Category
Purchase Amount
Payment Method
Purchase Date
Location

Project Workflow:

Data Collection
Imported dataset from CSV file
Data Cleaning
Removed missing values and duplicates
Corrected data types using Power Query
Data Analysis

Created DAX measures:
Total Sales
Total Customers
Average Purchase
Total Transactions
Sales per Customer
Dashboard Creation

Built interactive visualizations:
Sales by Category (Bar Chart)
Monthly Sales Trend (Line Chart)
Customer Demographics (Pie Chart)
Payment Method Analysis (Donut Chart)
Location Analysis (Map)

Key Insights:
The highest sales are observed in the top product category
Most customers prefer a specific payment method
Sales show variation across different months indicating trends
Customer demographics help in targeted marketing
